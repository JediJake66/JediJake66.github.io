Mystery Educator

This is a team project where we made a wen application for children ages 6-13. The user has an opportunity to learn about classic art, be given a random activity to do, or select a link to a number of fun games. We utilized agile and scrum methodologies to complete the project before the deadline. We used a MVC design to create our own API and used JavaScript, HTML, and CSS to wire in API's from the Metropolitan Art Museum and Bored.
